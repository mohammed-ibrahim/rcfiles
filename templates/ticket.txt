--guidelines

Test case name's should be very carefully selected.
public method which doesn't belong to interface must have description.
Log very carefully
Ensure that every assert has message along with it, No assert's without messages.
Run all the test classes.
New classes must have description over it.
Minimal amount of code changes for any monolith changes.
Copyright for new Classes
String literals, magic numbers
